<title>HTML</title><text>{{Infobox file format | name = HTML<br />{{small|{{nowrap|(HyperText Markup Language)}}}} | icon = | screenshot =  | extension = .html, .htm | mime = text/html | type code = TEXT | uniform type = public.html | owner = [[World Wide Web Consortium]] & [[WHATWG]] | genre = [[Markup language]] | container for =  | contained by = | extended from = [[Standard Generalized Markup Language|SGML]] | extended to = [[XHTML]] | standard = ISO/IEC 15445<br />[http://www.w3.org/TR/1999/REC-html401-19991224/ W3C HTML 4.01]<br /> [http://dev.w3.org/html5/spec/ W3C HTML 5] (draft) }}  '''Hyper Text Markup Language''' ('''HTML''') is a [[markup language]]<ref>{{cite web|url=http://infospace.ischool.syr.edu/2012/04/05/why-html-is-not-a-programming-language/|title=Why HTML is Not a Programming Language|publisher=Syracuse University|accessdate=27 June 2016}}</ref> for creating a [[webpage]]. In easier words, [[HTML]] is a kind of [[programming language]] that can make a new webpage. Webpages are usually viewed in a web browser. They can include writing, links, pictures, and even sound and [[video]]. HTML is used to mark and describe each of these kinds of content so the web browser can display them correctly. HTML also adds meta information to a webpage. Meta information is usually not shown by web browsers and is data ''about'' the web page, e.g., the name of the person who created the page. [[Cascading Style Sheets]] (CSS) is used to style HTML elements while [[JavaScript]] is used for website behavior and also changing the HTML and CSS.   HTML is made by the [[World Wide Web Consortium]] (W3C). There are several versions of HTML. As of September 2018, the current standard of HTML is dubbed [[HTML5|HTML 5]] and is specifically at version 5.2.  == Tags == HTML uses "elements" to let the browser know how a webpage is made of. Elements are shown as "tags" in the code, written with angle [[Bracket|brackets]]: {{code|lang=html|<example>}}. Tags ''usually'' come in pairs: an <u>opening</u> tag defines the start of a block of content and a <u>closing</u> tag defines the end of that block of content. There are many different kinds of tags, and each one has a different purpose. See '''[[HTML#Basic HTML tags|Basic HTML Tags]]''' below for tag examples.  Some tags only work in certain browsers. For example, the <code><nowiki><menuitem></nowiki></code> tag, which is used to make something appear when the person [[Click|presses]] the right button of the [[Computer mouse|mouse]], only works on the [[Mozilla Firefox]] browser.<ref>{{cite web | url=https://developer.mozilla.org/en-US/docs/Web/HTML/Element/menuitem | title=MDN Web Docs | date=2019-12-25 | access-date=2020-01-15 }}</ref> Other browsers simply ignore this tag and display the writing normally. Many web page creators avoid using these "non-standard" tags because they want their pages to look the same with all browsers.  == Example ==  Here is an example page in HTML with "[[Hello world program|Hello world!]]".<syntaxhighlight lang="html5"> <!DOCTYPE html> <html>   <head>     <title>Hi!</title>   </head>   <body>     <p>Hello world!</p>   </body> </html> </syntaxhighlight>The text between {{code|lang=html|code=<html>}} and {{code|lang=html|code=</html>}} describes the web page, and the text between {{code|lang=html|code=<body>}} and {{code|lang=html|code=</body>}} is the page content. {{code|lang=html|code=<title>This is a title</title>}} defines the browser page title.  ==Example HTML tags== Here are some example HTML tags: {{Refn|group=note|There is a full list of HTML elements here: <ref>{{cite web |title=HTML elements reference |url=https://developer.mozilla.org/en-US/docs/Web/HTML/Element |website=MDN |accessdate=6 August 2020}}</ref>}} {| class="wikitable sortable mw-collapsible mw-collapsed" !
