<title>Wikipedia:Mailing lists</title><text>[[Wikipedia]] has lots of ''mailing lists''. You can read them on the [[internet]] or by [[email]] or by [[NNTP]] using [[GMANE]]. [[Mailing list]]s show up in [[Google]] searches.  Old messages can be found at GMANE [http://www.gmane.org/] and MARC [http://marc.theaimsgroup.com/].   There is not a mailing list only for Simple English. You should probably use the English list or the international list.  == Project-specific == Administrators' mailing list, mainly used as a route of appeal for unblocks. All current administrators may subscribe to this list. *[[mail:simple-admins-l|List info page]]  The [[WP:Oversight|oversight]] team also have a mailing list for suppression requests. Only current oversighters may subscribe to this list. *[[mail:oversight-wp-simple|List info page]]  == General, project-wide mailing list == The general project-wide list is '''[[mail:wikipedia-l|Wikipedia-L]]'''.  It's a resource for Wikipedia authors. If you don't want to use Wikipedia (or [[m:|Meta]]) for questions, or if you'd like to talk to other [[Wikipedia:Wikipedians|Wikipedians]] outside the wiki format, then you can sign up for Wikipedia-L.  * [[mail:wikipedia-l|Wikipedia-L sign-up page]] * [[mailarchive:wikipedia-l|Wikipedia-L archive]] * [news://news.gmane.org/gmane.science.linguistics.wikipedia.misc Wikipedia-L via NNTP]  == English Wikipedia mailing list == Since Wikipedia-L developed into an extremely high traffic list, a separate mailing list for the English Wikipedia was created. For discussion of issues concerning the English wikipedia only, please sign up for '''<nowiki>WikiEN-L</nowiki>''' and keep Wikipedia-L for general policy discussions!   * [[mail:wikien-l|WikiEN-L sign-up page]] * [[mailarchive:wikien-l|WikiEN-L archive]] * [news://news.gmane.org/gmane.science.linguistics.wikipedia.english WikiEN-L via NNTP]  == Foundation mailing list == Foundation-l is for any cross-project issues, discussions and cooperation. It  is also for things that affect the Wikimedia foundation. Legal issues can be posted to this list.  * [[mail:foundation-l|Foundation-L sign-up page]] * [[mailarchive:foundation-l|Foundation-L archive]] * [news://news.gmane.org/gmane.science.linguistics.wikipedia.foundation Foundation-L via NNTP]  == Language-specific mailing lists == There are also language-specific mailing lists for Wikipedia in [[mail:wikida-l|Danish]], [[mail:wikinl-l|Dutch]], [[mail:wikieo-l|Esperanto]], [[mail:wikies-l|Spanish]], [[mail:wikifr-l|French]], [[mail:wikide-l|German]], [[mail:wikija-l|Japanese]], [[mail:wikipl-l|Polish]], [[mail:wikipt|Portuguese]] and [[mail:wikisv-l|Swedish]].  == Technical issues mailing list == And, there's a mailing list for technical discussions regarding development of the Wikipedia software: '''Wikitech-L'''.  * [[mail:wikitech-l|Wikitech-L sign-up page]] * [[mailarchive:wikitech-l|Wikitech-L archive]] * [news://news.gmane.org/gmane.science.linguistics.wikipedia.technical Wikitech-L via NNTP]  == Wikibooks mailing list == Mailing list for the [[wikibooks:|Wikibooks]] project.  * [[mail:textbook-l|Textbook-l sign-up page]] * [[mailarchive:textbook-l|Textbook-l archive]] * [news://news.gmane.org/gmane.org.wikimedia.textbook Textbook-L via NNTP]  Wikipedia also has a real-time chat channel. See [[meta:IRC channels]] for more info.  == Splitting up the Daily Digests == For Linux-Users there is a comfortable solution for splitting up the daily digest.
