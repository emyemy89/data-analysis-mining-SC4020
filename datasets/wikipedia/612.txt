Most tools and bots only want to know about disambig templates. But the interwiki bots consider set index articles to be "disambig similar", they allow interwiki links between set index articles here on the English Wikipedia and disambig pages on other Wikipedias. So the interwiki bots need to know about the set index templates.  <!-- Note!
