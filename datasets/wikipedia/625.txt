<title>Wikipedia:MediaWiki namespace</title><text>The '''MediaWiki namespace''' is a way to change the look of Wikipedia. You can change the words and text used for site links and text. For example, the "{{int:tagline}}" message which you can see at the top of each page is available at [[MediaWiki:Tagline]].  Only [[Wikipedia:Administrators|Administrators]] can change the MediaWiki namespace. If you want to make changes, please discuss them on [[Wikipedia talk:MediaWiki namespace|the talk page]].  For more help and information, see [[m:MediaWiki namespace]].  ==Related pages== * [[Special:Allmessages]] - A list of all the customisable MediaWiki messages, showing if they have been changed for this Wikipedia * [[Wikipedia:Template messages]] - a list of templates commonly used in this Wikipedia.</text>
